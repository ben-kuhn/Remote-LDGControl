LDG Tuner Flasher

ESP32 firmware for LDG AT-1000ProII / AT-600ProII auto-tuners

How to flash:

  1. Connect your ESP32 to this computer via USB
  2. Select your firmware variant below
  3. Click Install and follow the prompts
  4. After flashing, the ESP32 creates a WiFi AP: LDGConfig (password: configure)
  5. Connect to it — a captive portal opens for WiFi setup
Note: Requires Chrome, Edge, or Opera. Safari and Firefox are not supported.

Display Unit

Shack controller with BTT TFT35-SPI touchscreen. Web UI, MQTT, touch controls, and acts as the server endpoint a remote unit subscribes to.

Remote Unit

Headless. Full web server, MQTT, and tuner UART; can also subscribe to a Display Unit over SSE when configured. Use this for any non-display build.
