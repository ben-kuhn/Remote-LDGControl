LDG Tuner Flasher

ESP32 firmware for LDG AG-1000ProII / AG-600ProII auto-tuners

How to flash:

  1. Connect your ESP32 to this computer via USB
  2. Select your firmware variant below
  3. Click Install and follow the prompts
  4. After flashing, connect to the WiFi AP to configure
Note: Requires Chrome, Edge, or Opera. Safari and Firefox are not supported.

Display Unit

For units with BTT TFT35-SPI touchscreen. Includes web UI, MQTT, and touch controls.

Remote Unit

Headless unit for remote tuner locations. Connects to a display unit.

Basic Unit

Headless unit with web UI and MQTT. No remote link or display support.
